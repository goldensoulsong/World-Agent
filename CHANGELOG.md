

# 更新日志 (Changelog)

## [v0.02] - 2026-06-05

### 架构调整
* **代码层级解耦**：将项目重构为 `core`（核心逻辑）、`skill`（业务工作流）、`tool`（执行动作）三层架构。
* **技能按需加载**：引入动态挂载机制，大模型通过 `load_skill` 工具按需拉取 SOP，降低 Context 消耗。

### Prompt 优化
* **优化 ReAct 思维链**：更新 `MASTER_AGENT_PROMPT`，强化 4 步推理逻辑。
* **增加防御性限制**：严禁 Agent 在指令不明时盲目调用探索类工具进行试错。

### 功能增强
* **智能文档切分**：新增 `smart_chunking` 技能，支持基于文本结构动态切分多种格式的文档。
* **大文件预览**：优化 `read_file` 工具，读取大于 100KB 的文件时默认返回前 1000 字符的预览。
* **异常拦截**：修复读取二进制文件（如图片、PDF）时导致的 `UnicodeDecodeError` 崩溃。

### 安全与配置
* **隐私清理**：移除代码中打印环境变量（如 API_KEY）的终端调试语句。
* **规范追踪**：更新 `.gitignore`，不再追踪本地说明文档及测试模拟器文件。

## [v0.03] - 2026-06-05 (Nexus Web Edition)

### 🚀 第一部分：Web 交互引擎与可视化工作台
* **Web UI 重构**：基于 FastAPI 与 SSE 推送，全面升级为毛玻璃风格 Web 界面。
* **便捷交互**：左侧支持文件拖拽上传，右侧产出文件支持点击预览与下载。
* **纯净通信**：主聊天框隐藏繁杂思考过程，仅展示状态提示与最终正文。
* **监控面板 (HUD)**：
  * **Activity Log**：记录系统动作流水。
  * **Trace Log**：追踪底层 ReAct 调用链。
  * **Context Memory**：实时查看完整上下文。
* **设置中心**：新增网页配置面板，支持 API 热重载及一键拉取模型列表。
* **启动器**：重构全英文 `start.bat`，自动拉起浏览器。

### 🛡️ 第二部分：架构解耦与核心工具优化
* **工具分类解耦**：物理工具重组为 `network`、`sandbox`、`system` 三大模块。
* **沙盒安全加固**：通过 `os.path.commonpath` 严格限制路径，杜绝路径穿越。
* **执行工具调优**：优化 `clean_text` 与 `chunk_text` 工具的执行效率与容错率。

### 🧠 第三部分：强化工作流与心智升级
* **语义推断前置**：优化数据清洗 SOP，要求 Agent 执行前必须进行语境推演。
* **防盗版破译**：强化清洗能力，支持修复被拼音或特殊符号干扰的核心词汇。
* **全参数精洗拦截墙**：针对高成本的“全量精洗”工作流，设立强制口令核对拦截（必须回复“我确定使用全参数量精洗”）。
