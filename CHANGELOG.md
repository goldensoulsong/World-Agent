

# 更新日志 (Changelog)

## [v0.02] - 2026-06-05

### 架构调整
* **代码层级解耦**：将项目重构为 `core`（核心逻辑）、`skill`（业务工作流）、`tool`（执行动作）三层架构。
* **技能按需加载**：引入动态挂载机制，大模型通过 `load_skill` 工具按需拉取 SOP，降低 Context 消耗。

### Prompt 优化
* **优化 ReAct 思维链**：更新 `MASTER_AGENT_PROMPT`，强化 4 步推理逻辑。
* **增加防御性限制**：严禁 Agent 在指令不明时盲目调用探索类工具进行试错。

### 功能增强
* **智能文档切分**：新增 `smart_chunking` 技能，支持基于文本结构动态切分多种格式的文档。
* **大文件预览**：优化 `read_file` 工具，读取大于 100KB 的文件时默认返回前 1000 字符的预览。
* **异常拦截**：修复读取二进制文件（如图片、PDF）时导致的 `UnicodeDecodeError` 崩溃。

### 安全与配置
* **隐私清理**：移除代码中打印环境变量（如 API_KEY）的终端调试语句。
* **规范追踪**：更新 `.gitignore`，不再追踪本地说明文档及测试模拟器文件。

## [v0.03] - 2026-06-05 (Nexus Web Edition)

### 🚀 第一部分：全新 Web 交互引擎与可视化工作台
* **Web 化底层支持**：正式告别 CLI 命令行，引入 FastAPI 提供底层服务，采用 SSE 流式推送实现真正的打字机效果。
* **毛玻璃科幻美学**：全新手搓极度科幻、暗黑毛玻璃风格的主聊天界面，支持左右两侧隐形抽屉，直接管理 `workspace` 的输入与输出文件。
* **拖拽式无感交互**：支持将文件直接“拖拽”至左侧抽屉上传；右侧产出文件支持全局点击快捷预览与下载。
* **纯净状态化通信**：将繁杂的 `[Thought]` 和 `[Action]` 从主聊天流中剥离，主界面仅保留干净闪烁的“状态提示”，并在最后只打印正文。
* **极客监控舱 (HUD)**：新增悬浮控制模块，包含：
  * **Activity Log**：记录用户文件上传及系统动作流水账。
  * **Trace Log**：专门用于追踪大模型的原始 ReAct 脑电波与工具调用链，专供高阶查错使用。
  * **Context Memory**：实时查看大模型完整会话上下文（含 System Prompt 和参数）的记忆显微镜。
* **可视化设置中心 (Settings Hub)**：新增“齿轮插头”配置面板，支持动态修改 `API_KEY`、`BASE_URL` 并**瞬间热重载 Agent**，且集成了 `/v1/models` 标准协议一键拉取远端模型列表。
* **启动器重构**：重构 `start.bat` 为全英文、支持自动拉起浏览器的防乱码一键启动器。

### 🛡️ 第二部分：底层架构解耦与核心工具优化
* **工具包分类解耦**：将原来杂乱的物理工具进行了二次架构重组，解耦为 `network/`（网络层）、`sandbox/`（沙盒层）、`system/`（系统层），架构更加清晰。
* **终极物理沙盒防御**：重写底层物理操作工具，全量引入 `os.path.commonpath` 严格验证绝对路径，彻底封杀路径穿越（如 `../../`）风险。
* **执行侧工具调优**：深度优化了底层 `clean_text` 与 `chunk_text` 工具的执行效率与错误容忍度，确保大模型下达的指令能够更稳定地在物理沙盒中着陆。

### 🧠 第三部分：强化核心工作流与大模型心智升级
* **深度语义推断剖析**：全面改善数据清洗（Data Cleaning）的工作流 SOP。要求 Agent 必须具备语境推测的“前置思考”能力，不再盲目执行。
* **防盗版破译引擎**：在清洗工作流中赋予 Agent 甄别异常手段的“火眼金睛”，主动识破并修复拼音（如 `riyue`）或夹带干扰符（如 `精*神`）的核心词汇。
* **防手滑安全口令墙**：针对极度消耗 Token 账单的“全量大模型精洗”分支，在工作流中设立了绝对安全拦截墙。Agent 必须强制索要特定口令 `“我确定使用全参数量精洗”` 才允许放行执行。
