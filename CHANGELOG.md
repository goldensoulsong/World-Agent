# 更新日志 (Changelog)

## [v0.02] - 2026-06-05

### 架构调整
* **代码层级解耦**：将项目重构为 `core`（核心逻辑）、`skill`（业务工作流）、`tool`（执行动作）三层架构。
* **技能按需加载**：引入动态挂载机制，大模型通过 `load_skill` 工具按需拉取 SOP，降低 Context 消耗。

### Prompt 优化
* **优化 ReAct 思维链**：更新 `MASTER_AGENT_PROMPT`，强化 4 步推理逻辑。
* **增加防御性限制**：严禁 Agent 在指令不明时盲目调用探索类工具进行试错。

### 功能增强
* **智能文档切分**：新增 `smart_chunking` 技能，支持基于文本结构动态切分多种格式的文档。
* **大文件预览**：优化 `read_file` 工具，读取大于 100KB 的文件时默认返回前 1000 字符的预览。
* **异常拦截**：修复读取二进制文件（如图片、PDF）时导致的 `UnicodeDecodeError` 崩溃。

### 安全与配置
* **隐私清理**：移除代码中打印环境变量（如 API_KEY）的终端调试语句。
* **规范追踪**：更新 `.gitignore`，不再追踪本地说明文档及测试模拟器文件。
