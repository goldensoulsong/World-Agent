# World-Agent 更新日志 (Changelog)

## [v0.02] - 2026-06-05

本版本是一次**史诗级的底层架构重构**。我们将原本耦合在一起的执行动作与业务认知彻底剥离，赋予了 Agent 真正的“学习”和“查阅手册”的能力，标志着项目正式迈入了可无限扩展的智能体时代。

### 🚀 架构大升级 (Architecture Refactoring)
* **引入按需加载技能系统 (On-Demand Skills)**：将底层物理动作（Tools）与业务操作规范（Skills/SOP）进行了深度解耦。大模型现在不再把所有知识死记硬背在 Prompt 中，而是通过调用专门的 `load_skill` 工具，按需把“操作手册”下载到工作流中，彻底解决了 Context 溢出问题。
* **三层彻底解耦**：全面确立了 `core`（永远不变的思考引擎与底座）、`skill`（可随时插拔的业务工作流大脑）、`tool`（向外拓展的物理执行动作）的三层独立开源框架标准。

### 🧠 认知与思考逻辑进化 (Prompt Optimization)
* **打造超级 ReAct 主脑**：重构并融合了全新的 `MASTER_AGENT_PROMPT`。引入了极具深度的 4 步强制思维链（意图扫描 -> 现状与工具分析 -> 缺失定位与风险评估 -> 决断交互）。
* **防御性与克制机制**：严厉规避大模型的“幻觉”与“工具依赖症”。系统级禁止在指令模糊或信息不足时，盲目调用探测型工具（如全网搜索、大库检索）去乱试错，实现了真正的“三思而后行”与强制确认。

### ✨ 核心功能增强 (Feature Enhancements)
* **万能格式智能切分**：依托于新架构下的 `smart_chunking`（智能切块）技能，Agent 现在能自己“看懂”各种文档的结构。无论是用 `#` 号分割的笔记、双换行的散文，还是卷章结构的小说，它都能自主提议专属的正则表达式进行精准切割。
* **大文件安全“试吃”**：全面增强了 `read_file` 工具的防御性。当遇到超大文件时，不再粗暴报错，而是智能提取开头 1000 字符作为 `preview` 返回给大模型，让 Agent 拥有完美的“嗅探”和判断体验。
* **二进制文件防崩**：修复了在意外读取图片、音频等非纯文本文件时引发的 `UnicodeDecodeError` 系统崩溃问题。

### 🔒 开源安全与隐私 (Security & Privacy)
* **敏感信息清扫**：剔除代码逻辑中潜在的 API_KEY 与 BASE_URL 终端打印泄露风险。
* **规范化上传**：更新 `.gitignore` 黑名单，拦截个人联调日志与演示文件，保障纯粹的开源净土。
