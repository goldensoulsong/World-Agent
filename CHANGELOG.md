

# 更新日志 (Changelog)

## [v0.02] - 2026-06-05

### 架构调整
* **代码层级解耦**：将项目重构为 `core`（核心逻辑）、`skill`（业务工作流）、`tool`（执行动作）三层架构。
* **技能按需加载**：引入动态挂载机制，大模型通过 `load_skill` 工具按需拉取 SOP，降低 Context 消耗。

### Prompt 优化
* **优化 ReAct 思维链**：更新 `MASTER_AGENT_PROMPT`，强化 4 步推理逻辑。
* **增加防御性限制**：严禁 Agent 在指令不明时盲目调用探索类工具进行试错。

### 功能增强
* **智能文档切分**：新增 `smart_chunking` 技能，支持基于文本结构动态切分多种格式的文档。
* **大文件预览**：优化 `read_file` 工具，读取大于 100KB 的文件时默认返回前 1000 字符的预览。
* **异常拦截**：修复读取二进制文件（如图片、PDF）时导致的 `UnicodeDecodeError` 崩溃。

### 安全与配置
* **隐私清理**：移除代码中打印环境变量（如 API_KEY）的终端调试语句。
* **规范追踪**：更新 `.gitignore`，不再追踪本地说明文档及测试模拟器文件。

## [v0.03] - 2026-06-05 (Nexus Web Edition)

### 🚀 全新前端架构 (Nexus UI)
* **Web 化重构**：正式从 CLI 黑框框时代跃迁至 Web 时代，引入 FastAPI 提供底层服务，采用 SSE (Server-Sent Events) 流式推送 ReAct 心智链路。
* **毛玻璃美学设计 (Glassmorphism)**：全新手搓极度科幻、暗黑毛玻璃风格的主聊天界面，支持左右两侧隐形抽屉，直接操作 `workspace` 的输入与输出。
* **文件快捷交互**：支持将文件直接“拖拽”至左侧抽屉上传；右侧抽屉产出的文件变更为全局可点击的超链接，支持快捷一键查看/下载。
* **启动器重构**：全面重构 `start.bat` 为全英文无乱码的一键启动器，支持自动拉起浏览器并兼容 `uv` 高速模式与原生 `python` 模式。

### 📊 极客监控舱 (HUD Panels)
* **纯净状态化通信**：将繁杂的 `[Thought]` 和 `[Action]` 从主聊天流中剥离，主界面仅保留干净闪烁的“状态提示”，并在最后只打印正文。
* **实时动态追踪 (Activity Log)**：新增悬浮面板，记录用户文件上传及动作流水账。
* **Agent 脑电波追踪 (Trace Log)**：将剥离出的原始 ReAct 思考与工具调用链重定向至独立抽屉，专供高阶查错使用。
* **记忆显微镜 (Context Memory)**：新增实时查看大模型完整会话上下文（含 System Prompt 和所有参数）的功能界面。

### ⚙️ 系统配置中心 (Settings Hub)
* **可视化热重载**：告别手动修改 `.env`！新增前端齿轮插头面板，支持动态修改 `API_KEY`、`BASE_URL` 并瞬间热重载 Agent，完全无需重启服务器。
* **一键模型拉取**：集成 `/v1/models` 标准协议探测功能，允许通过前端一键扫描远端厂商的所有可用模型，并自动填充下拉菜单。

### 🛡️ 安全与技能升维
* **终极沙盒加固**：重写底层物理操作工具，引入 `os.path.commonpath` 进行绝对路径解析与校验，彻底封杀任何路径穿越（如 `../`）注入系统目录的可能。
* **深度语义剖析 (Data Cleaning)**：大幅强化清洗技能的 SOP，要求 Agent 必须具备“语境推测”、“防盗版短语破译还原（拼音/干扰符排查）”的独立思考能力。
* **绝对交互拦截**：针对极度消耗算力的“全参数量大模型精洗”操作，设立硬性安全口令墙。Agent 必须强制索要特定口令 `“我确定使用全参数量精洗”` 才允许放行执行。
