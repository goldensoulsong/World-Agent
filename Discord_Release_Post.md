@everyone 🎉 **跟大家分享一个我最近手搓的 AI 效率小工具 —— World Agent (世界观与小说处理助手)！** 🎉

**这是个啥？**
如果你平时喜欢看网文、搞同人创作、或者带团跑 TRPG，你一定遇到过这种折磨：从网上扒下来的几十万字小说/设定集，里面**全！是！广！告！**（各种夹杂全角拼音、防屏蔽符号的牛皮癣网址），而且排版乱七八糟。
所以我干脆自己写了个基于大模型 ReAct 架构的智能体，专门用来**自动化“切块”和“清洗”这些超长语料**！

**✨ 核心亮点：**
- 🤖 **全自动去牛皮癣**：不用自己写复杂的正则，你只需要用自然语言跟它聊天（比如：“帮我把这本书里的全角网址广告洗掉”），它就会自动生成正则表达式并从头到尾帮你清洗干净。
- 📚 **智能章节切块**：几十万字的纯文本，一句话让它按“第XX章”结构化切分，干干净净。
- 🛡️ **防爆内存保护**：底层加了防呆拦截，遇到几十 MB 的大文件会自动阻断，再也不会因为塞了太多字导致大模型爆显存/报错啦。
- 🚀 **小白专属开箱即用**：**不需要你懂 Python 也不需要配环境！** 解压后直接双击 `start.bat`，它会自动帮你装好虚拟环境。第一次运行粘贴一下你的 API Key（推荐用 DeepSeek 等便宜好用的模型），以后就能像聊天一样使唤它了。
