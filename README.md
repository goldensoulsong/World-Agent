# World Agent - Nexus Web Edition (v0.1) 📖

欢迎使用 **World Agent (v0.1)**。这是一个旨在帮助您自动化处理长篇小说、跑团设定以及文本清洗的智能助手。
从 v0.1 版本开始，项目完成了**底层的分布式重构**与**前后端彻底分离**，为您带来极具科幻感与沉浸感的操作体验！

---

## 🎉 v0.1 重大更新日志

本版本经历了底层架构的全面升级，从单体脚本正式迈向企业级应用架构：

1. **前后端彻底分离 (Vite + React)**：
   - 废弃了原有的单文件原生 JS 方案，前端全面迁移至 `frontend/` 目录，基于 Vite + React 构建。
   - 实现了丝滑的组件化 UI 渲染，引入了 System HUD (状态机追踪、原始 I/O 日志、上下文记忆实时监控)。
2. **LangGraph 引擎重构**：
   - 后端 Agent 核心全面升级为 **LangGraph** 状态机架构。
   - 实现了基于 Server-Sent Events (SSE) 的流式输出，完美解析大模型的 `<thought>` 标签并同步在前端渲染思考链和工具调用动画。
3. **引入持久化数据库 (PostgreSQL)**：
   - 引入 `database/` 模块，集成 SQLAlchemy 与 Alembic 数据迁移工具，为后续的复杂状态持久化（如 RAG 向量引用、用户账户隔离）打下坚实基础。
4. **全套 Docker 容器化支持**：
   - 新增 `docker-compose.yml`，一键拉起后端 API、前端静态资源代理与 PostgreSQL 数据库环境。
5. **分布式目录架构**：
   - 重构了原先的单体 `core/` 和 `tool/` 目录，按照分布式与多智能体的思路拆分为：
     - `common/`：通用底层基建（公用网络工具、文件工具、RAG 数据库基础模块）。
     - `agents/`：垂类智能体集群（当前已实装 `text_processing` 文本处理智能体）。

## 🎨 核心亮点 (Nexus UI)

1. **傻瓜式 Web 交互**：告别繁琐的命令行黑框框！通过浏览器直接与 Agent 对话，流式输出思考过程与结果。
2. **可视化管理与工作区**：
   - 所有的处理产物全局可见，实时同步 `workspace/input` 和 `workspace/output` 的文件状态。
3. **极客监控看板 (SYSTEM HUD)**：
   - 🔔 **日志 (Logs)**：忠实记录每一次你发给大模型的完整 Payload 与 AI 返回的原始文本。
   - ⚙️ **追踪 (Trace)**：纯净跟踪 LangGraph 状态机的 ReAct 调用链，清晰展示每一步的“思考 → 调用工具 → 工具返回”。
   - 🧠 **记忆 (Memory)**：实时查看大模型当前的完整记忆树。
4. **系统配置中心 (Settings Hub)**：
   - 网页端直连大模型 API 配置，修改后**瞬间热重载**，无需修改代码或重启服务器！
   - 动态拉取当前 API Key 支持的可用模型列表。

## 🛠️ 智能技能 (Skills)

World Agent 拥有按需加载的智能技能（SOP 手册）：

1. **智能文档切分 (Smart Chunking)**：支持处理几十万字的纯文本小说，基于文本结构动态切分。
2. **深度清洗勘探 (Data Cleaning)**：具备深度语义剖析能力，能够识破并修复被替换为拼音或干扰符的核心词汇。针对极度消耗 Token 的操作设有强制交互口令墙，彻底杜绝账单爆炸风险。
3. **安全沙盒 (Sandbox)**：所有的底层操作被绝对隔离在 `workspace` 目录下，彻底封杀路径穿越攻击。

## 🚀 快速入门指南

为了让所有人都能零门槛使用，本项目提供了一个**傻瓜式一键启动脚本** `start.bat`。

只需双击运行根目录下的 `start.bat`，即可看到交互菜单，支持两种环境：

### 方式一：[1] Docker 一键启动（强烈推荐）
最纯净、最稳定的体验。不仅能启动前后端，还会为您**自动配置 PostgreSQL 数据库**。
- **前置要求**：只需安装 [Docker Desktop](https://www.docker.com/products/docker-desktop/) 并保持后台运行。
- **执行过程**：脚本将自动拉起所有服务，完成后自动为您打开浏览器。

### 方式二：[2] 本地开发启动
如果您不想安装 Docker，也可以直接在系统里跑。脚本会**全自动**帮您完成前端构建和后端依赖安装！
- **前置要求**：需要安装 [Node.js](https://nodejs.org/) 和 [Python 3.10+](https://www.python.org/)。
- **执行过程**：脚本将自动执行 `npm install` 与 `npm run build` 打包前端，然后自动创建 `.venv` 虚拟环境并安装 Python 依赖，最后启动服务并打开浏览器。

### 初始配置步骤

1. 在网页左侧的 **SYSTEM HUD** 中点击 **🔌 配置** 进入 Settings Hub。
2. 输入您的 `API_KEY` 和 `BASE_URL`（支持 DeepSeek 等主流大模型）。
3. 点击“🔄 拉取模型”，选择模型后点击“保存并热重载”。
4. 在聊天框直接下达自然语言指令！

## 💻 开发者架构说明

本项目采用了 **按需加载 (On-Demand Loading)** 与 **LangGraph 状态机** 相结合的智能体架构。

- `server.py`：FastAPI 网关，处理 SSE 流式推送和前端静态资源挂载。
- `agents/`：多智能体目录。当前主力为 `text_processing` 智能体。
  - `agent.py`：定义了 LangGraph 的节点（执行模型、执行工具）和边（路由判断）。
  - `skills/`：垂直领域的场景化 SOP 大脑。大模型会根据触发条件，自主调用 `load_skill_sop` 动态加载对应技能的详细规则。
  - `tools/`：该智能体专属的沙盒物理工具。
- `common/`：全局共享能力库。
  - `tools/`：网络搜索、文件读写等基础系统能力。
  - `rag/`：向量数据库基建（与 PostgreSQL / PGVector 交互）。
