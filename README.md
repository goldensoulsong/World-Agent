# World Agent - Nexus Web Edition 📖

欢迎使用 **World Agent (v0.03)**。这是一个旨在帮助您自动化处理长篇小说、跑团设定以及文本清洗的智能助手。
从 v0.03 版本开始，项目正式迈入 **Web 时代**，为您带来极具科幻感的毛玻璃 UI 界面与傻瓜式交互体验！

无论您是处理同人设定，还是清理包含防盗版乱码的网络小说，都可以使用 World Agent 高效完成。

---

## 🎨 核心亮点 (Nexus UI)

1. **傻瓜式 Web 交互**：告别繁琐的命令行黑框框！通过浏览器直接与 Agent 对话，流式输出思考过程与结果。
2. **可视化拖拽与管理**：
   - **左侧输入舱 (Input Zone)**：直接拖拽 TXT/JSON 文件即可上传。
   - **右侧输出舱 (Output Zone)**：所有的处理产物全局可见，点击文件名即可在浏览器中快捷预览或一键下载。
3. **极客监控看板 (HUD Panels)**：
   - 🔔 **Activity Log**：记录每一次文件操作与系统动态。
   - ⚙️ **Trace Log**：纯净跟踪大模型的底层 ReAct 调用链，不污染主对话框。
   - 🧠 **Context Memory**：实时查看大模型当前的完整记忆树。
4. **系统配置中心 (Settings Hub)**：
   - 网页端直连大模型 API 配置，修改后**瞬间热重载**，无需修改代码或重启服务器！
   - 支持一键拉取远端可用模型列表，告别手敲模型名称。

---

## 🛠️ 智能技能 (Skills)

1. **智能文档切分 (Smart Chunking)**：支持处理几十万字的纯文本小说，基于文本结构动态切分。
2. **深度清洗勘探 (Data Cleaning)**：
   - **防盗版破译**：具备深度语义剖析能力，能够识破并修复被替换为拼音（如 `riyue`）或干扰符（如 `精*神`）的核心词汇。
   - **安全拦截**：针对极度消耗 Token 的全参数量大模型精洗，设有强制交互口令墙，必须获得您的特定指令才允许执行，彻底杜绝账单爆炸风险。
3. **安全沙盒 (Sandbox)**：所有的底层操作被绝对隔离在 `workspace` 目录下，彻底封杀任何路径穿越攻击。

---

## 🚀 快速入门指南

无需手动配置复杂的开发环境，只需执行以下步骤即可开始您的赛博办公：

1. **一键启动**：双击运行项目根目录下的 `start.bat` 文件。它会自动为您安装 Web 依赖、构建虚拟环境，并**自动唤醒您的浏览器**进入工作台。
2. **配置大模型接入点**：
   - 在网页右上角点击 🔌 (插头图标) 进入 **Settings Hub**。
   - 输入您的 `API_KEY` 和 `BASE_URL`（支持 DeepSeek 等主流大模型）。
   - 点击“🔄 拉取模型”，选择模型后点击“保存并热重载”。
3. **下达指令**：在正中央的聊天框直接输入自然语言即可！
   
   **指令示例：**
   > “把我刚才拖进去的小说切一下块，建个新项目叫 斗罗大陆2。”
   > “帮我清洗一下刚才的文本，里面有很多防盗版的拼音乱码，仔细破译一下。”

*(注：系统会在您第一次启动 `start.bat` 时，自动为您创建 `workspace/input` 和 `workspace/output` 工作区目录。)*

## 💻 开发者架构说明

本项目采用了业界前沿的 **按需加载 (On-Demand Loading)** 与 **三层解耦** 智能体架构。如果您希望进行二次开发，请遵循以下规范：

- `core/`【核心发动机层】：包含大模型心智 Prompt 与 ReAct 循环路由。
- `tool/`【物理工具层】：供 Agent 调用的系统执行动作（包含 `sandbox/` 沙盒安全隔离工具和 `system/` 系统工具）。
- `skill/`【技能工作流层】：垂直领域的场景化 SOP 大脑。为了防止 Token 浪费，大模型会根据触发条件，自主调用 `load_skill` 动态下载 SOP 手册。
