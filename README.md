# World Agent - 小说与设定集处理助手 📖

欢迎使用 **World Agent**。这是一个旨在帮助您自动化处理长篇小说、跑团设定以及文本清洗的智能助手。

无论您是处理同人设定，还是清理包含乱码的网络文本，都可以使用 World Agent 高效完成。

---

## 🛠️ 核心功能特点

1. **自动切块排版**：支持处理几十万字的纯文本小说，自动按“第XX章”进行结构化切分与排版。
2. **智能抽样与广告过滤**：自动在文本的头部、中部、尾部进行深度抽样，识别并清洗防屏蔽的特殊网址或符号广告。
3. **大模型深度清洗**：针对无规律的杂乱文本或乱码，支持调用大模型进行并发全量清洗，精准提取有效内容。（⚠️ **警告**：全量清洗会消耗大量 API Token，百万字的文本可能耗费相应数量的 Token，请酌情使用）
4. **规范化输出**：所有处理结果均会分类保存在 `output/项目名称/` 目录下，保持工作区整洁。
5. **内存管理与限流保护**：底层实现了分块读取拦截机制，防止超大文件导致内存溢出，并内置高并发请求的限流保护。

---

## 🚀 快速入门指南

无需手动配置复杂的开发环境，只需执行以下步骤即可使用：

1. **一键启动**：双击运行项目目录下的 `start.bat` 文件。程序会自动检测并配置所需的 Python 环境与依赖库。
2. **配置 API 密钥**：首次运行时，程序会引导您输入 `API_KEY`（如 DeepSeek 等大模型的密钥）以及 `BASE_URL`。配置完成后，系统会自动生成 `.env` 文件保存您的设置。
3. **下达指令**：程序启动后，您可以通过自然语言直接与 Agent 对话。
   
   **指令示例：**
   > “帮我读取 `素材/斗罗大陆.txt`，用切块工具把它切好，项目名叫 斗罗大陆2。”
   > “帮我清洗一下刚才切好的数据，用清洗工具洗掉里面的网址广告。”

## 📁 项目目录结构

项目的核心结构与对应的功能说明如下：

```text
World-Agent/
├── .env                   # 本地环境变量配置文件（首次运行后自动生成，存储 API_KEY 等）
├── core/                  # 核心架构层
│   ├── prompts.py         # 系统提示词（System Prompts）配置
│   └── react_agent.py     # ReAct Agent 核心控制器（负责解析意图与调度工具）
├── tool/                  # 扩展工具箱
│   ├── chunk_text/        # 长文本切块排版工具
│   ├── clean_text/        # 文本去广告清洗、大模型深度清洗工具
│   ├── json_to_txt/       # JSON 转 TXT 格式转换工具
│   ├── list_dir/          # 目录结构读取工具
│   ├── read_chunk/        # 分块文件读取工具
│   ├── read_file/         # 基础文件读取工具
│   ├── web_search/        # 联网搜索工具
│   ├── write_file/        # 文件写入/保存工具
│   └── __init__.py        # 工具统一注册与导出模块
├── main.py                # 主程序入口（包含交互式配置引导）
├── start.bat              # Windows 一键启动脚本（全自动准备环境）
├── requirements.txt       # Python 依赖库清单
├── pyproject.toml         # 项目打包与包管理器配置
└── README.md              # 项目说明文档
```

## 💻 开发者架构说明

如果您熟悉 Python 并希望进行二次开发，项目的核心代码结构非常清晰：
- **核心控制器**：`main.py` 和 `core/react_agent.py`（基于纯手写的 ReAct 架构构建）。
- **工具箱**：`tool/` 文件夹（扩展新功能只需在此目录添加相应的 Python 脚本即可）。
- **系统提示词**：`core/prompts.py`（控制 Agent 行为和设定的指令集）。
