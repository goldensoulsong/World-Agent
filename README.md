# World Agent - 小说与设定集处理助手 📖

你好！欢迎使用 **World Agent**。这是一个专门为了帮你自动化处理长篇小说、跑团设定以及文本清洗而打造的 AI 小管家。

无论你是想处理同人设定，还是想清理爬下来的满屏乱码的语料，把它丢给这个智能体就对了！

---

## 🛠️ 它可以帮你做什么？

1. **自动切块排版**：当你有一本几十万字的纯文本小说时，只要告诉它，它就会自动帮你按“第XX章”进行完美的结构化切分。
2. **智能抽样与去牛皮癣**：网文里总是夹杂着防屏蔽的“全角拼音网址”、特殊符号广告？AI 会自动在全书头部、中部、尾部进行“深度抽样”，找出最狡猾的广告特征，然后一秒钟全量清洗！
3. **大模型究极精洗**：遇到毫无规律的“作者碎碎念”或是乱码？它自带“大模型并发全量清洗工具”，哪怕再顽固的牛皮癣，也可以直接开大招让大模型逐字剥离。（⚠️ **警告**：全量精洗会极度消耗 API Token，一本几百万字的小说可能耗费几百万 Token，请酌情使用）
4. **强迫症福音**：所有的处理结果都会整整齐齐地保存在 `output/你的项目名/` 文件夹下，绝不把你的文件夹搞得一团糟。
5. **安全防爆防呆**：不用担心 AI 傻乎乎地一口气读取千万字撑爆内存，程序底层有强大的拦截保护机制，同时内置高并发请求限流保护。

---

## 🚀 极简使用说明（小白必读）

你**完全不需要**懂代码，也不用自己配环境，只要按下面三步走：

1. **一键启动**：在文件夹里找到 `start.bat` 文件，双击运行。程序会自动帮你搞定所有复杂的 Python 依赖。
2. **配置钥匙**：第一次打开黑框框时，系统会找你要一个 `API_KEY`（比如 DeepSeek 的密钥），粘贴进去敲回车即可（以后它会偷偷记住，不用再填啦）。
3. **下达指令**：程序启动后，你就可以像老板使唤秘书一样跟它聊天了！
   
   **你可以直接这么对它说：**
   > “帮我读取 `素材/斗罗大陆.txt`，用切块工具把它切好，项目名叫 斗罗大陆2。”
   > “帮我清洗一下刚才切好的数据，用清洗工具洗掉里面那种带着全角数字的网址广告。”

## 📁 项目目录结构

为了方便大家查阅，下面是本项目的核心结构与对应的功能批注：

```text
World-Agent/
├── core/                  # 核心架构层
│   ├── prompts.py         # 系统提示词（用于给 AI "洗脑"的指令集）
│   └── react_agent.py     # ReAct Agent 智能体大脑总控（解析意图、调度工具）
├── tool/                  # 扩展工具箱（AI 技能包）
│   ├── chunk_text/        # 长文本智能切块排版工具
│   ├── clean_text/        # 文本去牛皮癣清洗、大模型究极精洗工具
│   ├── json_to_txt/       # JSON 转 TXT 格式转换工具
│   ├── list_dir/          # 目录结构读取工具
│   ├── read_chunk/        # 分块文件读取工具
│   ├── read_file/         # 基础文件读取工具
│   ├── web_search/        # 联网搜索工具
│   ├── write_file/        # 文件写入/保存工具
│   └── __init__.py        # 工具统一注册与导出模块
├── main.py                # 主程序入口（包含交互式新手配置引导）
├── start.bat              # Windows 一键启动脚本（纯英文，全自动准备环境）
├── requirements.txt       # Python 运行库依赖清单
├── pyproject.toml         # 项目打包与包管理器配置
└── README.md              # 项目说明文档
```

## 💻 给折腾党的架构提示

如果你懂点 Python，想要自己加新功能，代码其实非常好懂：
- **大脑总控**：`main.py` 和 `core/react_agent.py`（最纯粹的手写 ReAct 架构）
- **工具箱**：`tool/` 文件夹（想让 AI 会新的技能？直接在这里加 Python 脚本就行）
- **咒语本**：`core/prompts.py`（用来给 AI 洗脑的系统提示词）
